- This app is for a tire store chain. It allows them to add store locations in the stores tab, add employees and what store they work at, and add items and how much they cost.

- You can also filter employees based on what store they work at.

- Stores can then order a quantity of an item to there store, and they can then track the order and see when it is pending, outbound, or delivered.

Ways you can filter orders:

  • Pending orders
  • Outbound orders
  • Delivered orders
  • Orders for a set store
  • Orders for a set item
